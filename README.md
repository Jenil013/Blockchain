# 🔗 Blockchain in Python

A simple blockchain implementation in Python.
This project is built for learning purposes and demonstrates the fundamentals of how blockchains work.

---

## 🚀 Features
- Create a blockchain with blocks and transactions
- Proof-of-Work algorithm
- Consensus mechanism (resolve conflicts between nodes)
- Simple Flask API to interact with the blockchain
- Unit tests for reliability
